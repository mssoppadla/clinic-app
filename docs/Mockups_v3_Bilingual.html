Layout adjustments this surfaces (and how we handle them):
  • Labels become two lines → set min-heights on buttons/cards and use line-height ~1.25 so nothing clips; never truncate the Malayalam line.
  • Malayalam runs longer & taller than English → fluid widths, wrap-friendly chips/pills, no fixed-height rows.
  • Bundle a Malayalam-capable font (e.g., Noto Sans Malayalam) so glyphs render on every device/kiosk.
  • Buttons hold English (bold) + Malayalam (lighter) stacked → keep tap targets ≥44px; senior kiosk uses larger type.
  • Numbers/tokens/time stay one line; only the descriptive label is bilingual.
  • It's additive: turning Malayalam OFF simply hides the second line — English layout is unchanged (backward-compatible).
  • Not RTL — no mirroring needed; just vertical space.
